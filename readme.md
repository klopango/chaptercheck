# Bible Chapters
A CLI & NodeJS api to print some bible chapters.

> I use the indonesian version while making this module.

## CLI Usage
```
npm i -g biblechapters
chapters
```

## API
```javascript
const chapters = require('biblechapters');
chapters("kejadian")
```
